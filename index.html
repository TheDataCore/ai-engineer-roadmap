Recommended Capstone Project

Production RAG API with Evals & Cost Logging

Ingest a document corpus → embed & store in a vector DB → expose a streaming query endpoint → return structured JSON outputs → log every call with cost + latency → run automated evals on responses. This proves RAG, structured outputs, deployment, observability, and evals — everything an employer cares about.
